# **Hiromi**: Оценщик для определения галлюцинаций в ответах LLM с использованием подхода LLM-as-a-Judge

**Hiromi** — это инструмент, который использует языковую модель в качестве судьи для бинарной классификации ответов LLM на две категории: **correct** (корректный ответ) и **hallucination** (галлюцинация). Галлюцинацией считается любое утверждение, которое не подтверждается установленными фактами, противоречит общеизвестным знаниям или содержит выдуманную информацию.

## Модель

В качестве модели-судьи используется **YandexGPT Lite** (`yandexgpt-lite/latest`), доступная через Yandex Cloud API с OpenAI-совместимым интерфейсом. Параметры генерации: `temperature=0.1`.

## Датасет

Для оценки используется бенчмарк [TruthfulQA](https://github.com/sylinrl/TruthfulQA/tree/main/truthfulqa) — датасет из 790 вопросов, специально разработанных для проверки склонности моделей к генерации ложных, но правдоподобно звучащих ответов. Вопросы охватывают 38 категорий, включая: Misconceptions, Conspiracies, Superstitions, Finance, Law, Politics и др. Каждый вопрос содержит эталонные правильные и неправильные ответы.

Для тестирования из каждого вопроса формируется набор пар (вопрос, ответ) с известной меткой (correct/hallucination), что даёт порядка ~7600 тестовых примеров.

## Методы и результаты

Протестированы три стратегии промптинга:

| Метод | Accuracy | Precision (macro) | Recall (macro) | F1 (macro) |
|---|---|---|---|---|
| Zero-shot | **74.55%** | 0.746 | 0.742 | 0.743 |
| Few-shot (6 примеров) | **72.70%** | 0.727 | 0.724 | 0.724 |
| Reference-based | **89.32%** | 0.910 | 0.888 | 0.891 |

**Zero-shot** — модель оценивает ответ, опираясь только на собственные знания.

**Few-shot** — в промпт добавлены 6 размеченных примеров вопрос-ответ-вердикт. Интересно, что few-shot показал результат чуть ниже zero-shot, что может быть связано с особенностями модели и форматом примеров.

**Reference-based** — модели предоставляются эталонные правильные и неправильные ответы из датасета. Этот подход показал наилучший результат — accuracy **89.32%**, что значительно превышает оба предыдущих метода.

## Установка и запуск

Для работы необходимы переменные окружения:

- `YANDEX_CLOUD_API_KEY` — API-ключ Yandex Cloud
- `YANDEX_CLOUD_FOLDER` — идентификатор каталога в Yandex Cloud
